# Requirements

- [X] Docker
- [X] docker-compose

## Webs
 - [X] Webgoat: localhost:8080/WebGoat
 - [X] DVWAP: localhost:8081
 - [X] bwapp: localhost:8082/install.php
 - [X] Juice-Shop: localhost:8083

## Usage mode.

```bash
git clone https://github.com/MoisesTapia/TWAPT
cd TWAPT
docker-compose up -d

```
